# BAD-BLE-USB

## 📡 Description

This repository contains IR and RF files that can be used with the **Bruce firmware**.

The files are intended for use with compatible devices running Bruce firmware, allowing you to send, test, or replay signals.

---

## 📂 Contents

* **IR/**
  Infrared signal files (e.g. `.ir`) for various devices

* **RF/**
  Radio frequency related files

* **BAD USB_BLE/**
  Scripts and payloads for BLE/USB-based interactions

---

## ⚙️ Usage

1. Install the **Bruce firmware** on your device
2. Copy the desired files to your device storage
3. Load or execute them through the firmware interface

Make sure your device supports the specific signal type (IR or RF).

---

## ⚠️ Disclaimer

These files are provided for **educational and testing purposes only**.

Use them responsibly and only on devices you own or have permission to interact with.
