# BAD-BLE-USB

This repository contains a collection of **Bad USB / BLE scripts** designed for use with **iPhone and PC systems**.

## ⚠️ Warning

Some of the scripts in this repository can be **dangerous**. They may:

* Modify system settings
* Execute commands without user consent
* Cause unintended behavior or damage

**Use at your own risk.**

## ❗ Important Notice

These scripts are provided for **educational and research purposes only**.
If you do not fully understand what a script does, **do NOT run it**.

Testing these scripts on your main device is strongly discouraged.
Always use a **controlled environment** (e.g., virtual machines or test devices).

## 💻 Compatibility

* iPhone (via BLE-based interactions)
* Windows / PC systems
